# ClientSpoofer
This mod changes your client's brand name so that servers can't tell which client you're using.
## Supported platforms
| Version        | Fabric | Forge | NeoForge |
|----------------|--------|-------|----------|
| 1.21.4         | ✅      | ❌     | ❌        |❌|
| 1.20.1-1.20.3  | ❌      | ❌     | ❌        |❌|
| 1.20 and lower | ❌      | ❌     | ❌        |❌|
